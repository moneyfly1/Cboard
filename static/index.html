🚀 XBoard 静态文件服务

此页面显示可用的静态资源文件

🎨 CSS 样式文件

  • main.css - 主要样式文件

⚡ JavaScript 文件

  • utils.js - 工具函数库

🖼️ 图片文件

  • logo.svg - XBoard Logo
  • favicon.ico - 网站图标

📚 使用说明

在 HTML 中引用：

<!-- CSS 文件 -->
<link rel="stylesheet" href="/static/css/main.css">

<!-- JavaScript 文件 -->
<script src="/static/js/utils.js"></script>

<!-- 图片文件 -->
<img src="/static/images/logo.svg" alt="XBoard Logo">

在 Vue 组件中使用：

<template>
  <div>
    <img src="/static/images/logo.svg" alt="Logo">
  </div>
</template>

<script>
export default {
  mounted() {
    // 使用工具函数
    if (window.XBoardUtils) {
      window.XBoardUtils.showNotification('页面加载成功！', 'success');
    }
  }
}
</script>
